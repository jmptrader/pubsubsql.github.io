What is PubSubSQL?

PubSubSQL is an in-memory relational database written in Go with an integrated Publish-Subscribe notification pattern - no more periodic polling or implementing database triggers! Subscribe any of your web or backend services to changes in the database!

PubSubSQL can also use MySQL as a backend, so you don't have to change your existing analytical services code!
